# Local SEO Checklist — Ramilography (NJ / NYC)

The on-page SEO is handled in code (service pages, schema, image sitemap, etc.).
**This file is the off-site work — the part only Ramil can do, and it's the
highest-ROI of all.** For a local photographer, Google's map results and reviews
drive more leads than the website itself.

Legend: 🔴 do first · 🟡 soon · 🟢 ongoing

## 1. Google Business Profile (GBP) — 🔴 the #1 priority
This is what shows up in Google Maps and the "near me" pack. Free.

- [ ] Go to **google.com/business** and create/claim **Ramilography**.
- [ ] Category: **Photographer** (primary). Add secondary categories that fit:
      Portrait Photographer, Event Photographer, Photography Studio.
- [ ] **Service area:** since you travel, set it as a *service-area business* —
      list Newark + the NJ towns you cover + New York City. (You don't have to
      publish a home address.)
- [ ] **Phone:** (862) 414-4948 · **Website:** https://ramilography.com
- [ ] **Hours:** set realistic contact hours (even "by appointment").
- [ ] **Services:** add Portraits, Couples, Family, Events, Sports with the
      "from" prices and short descriptions (match the website).
- [ ] **Photos:** upload 15–25 of your best — this hugely affects ranking and
      clicks. Add a logo + a cover image.
- [ ] Turn on **messaging** so leads can text you from the profile.
- [ ] Verify the profile (postcard / phone / video — Google will prompt).
- [ ] **NAP consistency:** make sure Name / Phone are written *identically*
      everywhere (website footer, GBP, Instagram, directories). Google cross-checks.

> Tip: once verified, **add me (the AI) as a Manager** under Settings → People,
> and I can keep it optimized + post updates.

## 2. Reviews — 🔴 second priority (drives ranking + trust)
- [ ] After GBP is verified, grab your **review link**: GBP dashboard → "Ask for
      reviews" → copy the short link (looks like `g.page/r/...`). Send it to me —
      I'll add a one-tap **"Leave a Review"** button to the site + booking
      follow-ups.
- [ ] Ask **every** happy client right after delivery. A simple text:
      *"So glad you loved the photos! If you have 30 seconds, a quick Google
      review really helps me out: [link]"*
- [ ] Goal: **5 reviews fast**, then a steady trickle. Reply to every one.
- [ ] Screenshot your best reviews and send them to me for a testimonials
      section on the site.

## 3. Search Console + analytics — 🟡 so we can measure
- [ ] Set up **Google Search Console** (search.google.com/search-console),
      verify ownership of ramilography.com (DNS or HTML tag — I can add the tag).
- [ ] Submit the sitemap: `https://ramilography.com/sitemap.xml`.
- [ ] Send me your **GA4 Measurement ID** (`G-XXXX`) and **Meta Pixel ID** —
      I'll switch on the analytics that's already wired into the site.
- [ ] (Optional) Bing Webmaster Tools — quick win, import from Search Console.

## 4. Local citations / directories — 🟡 consistency = authority
Create/claim listings with the **exact same** name, phone, and URL:
- [ ] Apple Maps (Apple Business Connect) · Bing Places · Yelp
- [ ] Photographer-specific: Thumbtack, The Knot / WeddingWire (even non-wedding),
      Bark, Google's local competitors' directories.
- [ ] Instagram + Facebook: make sure the bio link is ramilography.com and the
      name/handle match.

## 5. Ongoing content — 🟢 compounds over time
- [ ] Post to GBP (offers, recent work) ~weekly — like mini social posts.
- [ ] Add recent shoots to the portfolio so there's fresh, indexable imagery.
- [ ] Later: a simple "recent sessions" page/blog to capture long-tail searches
      ("maternity photos in [NJ town]"). I can build this when ready.
